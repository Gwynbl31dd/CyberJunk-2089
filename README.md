# CyberJunk-2089

Unity project Demo

This is a Unity projet in progress, inspidered by Cyber punk, neo tokio, Blader runner and so on.

# Keys

* A - D / <- -> : Move the camera left and right
* Mouse right click : Move to position.
* Mouse left click : Attack/interact.
* Shift : Walk
* F2 : Quick Save
* F3 : Quick Load
* TAB : Open inventory
